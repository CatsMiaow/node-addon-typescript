# node-addon-typescript

Node.js WebAssembly and C++ Addons TypeScript Example

## Configure

```sh
npm ci
npm run addon:init
```

## Build

```sh
npm run addon:build # cpp
npm run asbuild # assembly
```

## Run

```sh
npm run ts src/index
# OR
npm start
```
